Halo, kali ini saya akan memberikan sedikit dokumentasi mengenai API Daerah Indonesia. Apa itu API Daerah Indonesia ? API Daerah Indonesia adalah API yang berisi data-data Provinsi, Kota / Kabupaten, Kecamatan dan Kelurahan. Data ini saya ambil dari Github ( https://github.com/edwardsamuel/Wilayah-Administratif-Indonesia ), jika dilihat dari Github tersebut data terakhir ditarik per-tanggal 11th January 2018.
